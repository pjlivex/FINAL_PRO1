About Blockcraft

```

Blockcraft is a browser-based sandbox voxel game that allows players to explore, build, and interact with a block-based world directly from their web browser. Inspired by classic block-building games, Blockcraft focuses on creativity, simplicity, and accessibility.

Unlike traditional games that require downloads or installations, Blockcraft runs entirely online. This means you can start playing instantly without any setup, making it easy for anyone to jump in and enjoy the experience.

The game is built using modern web technologies such as Three.js, delivering smooth performance and interactive 3D environments. Players can place and remove blocks, experiment with different materials, and create their own structures freely.

Our goal is to provide a fun and lightweight sandbox experience that works across devices, including desktops and mobile browsers. Whether you want to relax and build or experiment with creative ideas, Blockcraft offers an open environment to do so.

This project is continuously improving, with future updates planned to enhance gameplay, add new features, and improve performance.
